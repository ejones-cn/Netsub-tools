# IP子网切分工具 v1.0.0

一个灵活的IP子网切分工具，可以将指定子网从父网络中切分出来，并展示各网段的详细信息。

## 功能特性

- ✅ 支持将指定子网从父网络中精确切分
- ✅ 展示切分网段的详细信息（网络地址、子网掩码、广播地址等）
- ✅ 显示剩余网段的完整列表和详细信息
- ✅ 提供Web界面和Windows GUI两种界面
- ✅ 支持任意CIDR格式的IP网段
- ✅ 可视化显示网段分布情况
- ✅ 支持导出剩余网段列表

## 安装依赖

### 核心功能
核心功能仅依赖Python标准库的`ipaddress`模块，无需额外安装。

### Web界面
Web界面需要安装Flask：
```bash
pip install flask
```

### Windows GUI界面
Windows GUI界面使用Python内置的tkinter库，无需额外安装。

## 使用方法

### Web界面

```bash
python web_app.py
```

然后在浏览器中访问：`http://localhost:5000`

或者使用批处理脚本直接运行：

```bash
.\run_web_app.bat
```

### Windows GUI界面

```bash
python windows_app.py
```

或者使用批处理脚本直接运行：

```bash
.\run_app.bat
```

也可以使用编译好的可执行文件（位于`dist`目录下）：

```bash
.\dist\IP子网分割工具.exe
```

## 工具原理

1. **输入验证**：检查要切分的子网是否属于父网段
2. **递归切分**：使用递归算法将父网段逐步切分为更小的子网
3. **子网排除**：排除指定的子网，保留剩余部分
4. **子网聚合**：将相邻的子网合并，减少输出数量
5. **信息展示**：生成详细的子网信息报告

## 文件说明

- `ip_subnet_calculator.py`：核心计算模块，包含所有IP子网计算功能
- `web_app.py`：Web界面，提供直观的操作体验
- `windows_app.py`：Windows GUI界面，提供桌面应用体验
- `run_app.bat`：Windows GUI界面运行脚本
- `run_web_app.bat`：Web界面运行脚本
- `IP子网分割工具.spec`：PyInstaller编译配置文件
- `IMPLEMENTATION_SUMMARY.md`：实现总结文档
- `README.md`：项目说明文档
- `requirements.txt`：项目依赖文件
- `setup.py`：项目安装配置文件

## 技术特点

- **使用标准库**：核心功能仅使用Python标准库的`ipaddress`模块
- **高效算法**：采用递归切分和子网聚合算法，确保结果准确高效
- **友好界面**：提供Web界面和Windows GUI界面两种选择，满足不同使用场景
- **详细信息**：展示每个网段的完整信息，包括可用地址数量
- **可视化展示**：使用图表直观显示网段分布情况
- **便捷操作**：提供批处理脚本和编译好的可执行文件，方便快速使用

## 注意事项

1. 确保输入的IP网段格式正确（如：10.0.0.0/8）
2. 要切分的子网必须是父网段的有效子网
3. 对于大型网络（如/8），剩余网段数量可能非常多
