# IP子网切分工具 v1.2.0

**IP子网切分工具** - 让IP地址管理变得简单高效！

## 📋 项目概述

IP子网切分工具是一个功能强大、易于使用的网络工具，旨在帮助网络管理员和工程师快速、准确地进行IP地址规划和子网划分。该工具支持CIDR表示法，能够自动计算子网信息，并提供友好的用户界面。

## ✨ 功能特性

- **CIDR输入支持**：直接输入CIDR格式的IP地址（如192.168.1.0/24）
- **子网切分**：将一个大网段划分为多个小网段
- **参数计算**：自动计算网络地址、广播地址、可用IP范围、子网掩码等
- **直观显示**：清晰展示每个子网的详细信息
- **多种界面**：支持Web界面和Windows GUI界面
- **批量处理**：支持一次性输入多个网段进行计算
- **结果导出**：可将计算结果导出为文本格式

## 🛠️ 工具原理

本工具基于IPv4地址的子网划分原理，使用Python实现了以下核心功能：

1. **IP地址解析**：将输入的IP地址转换为二进制格式
2. **子网掩码计算**：根据CIDR前缀长度计算子网掩码
3. **网络地址计算**：通过IP地址与子网掩码的AND运算得到网络地址
4. **广播地址计算**：通过网络地址与子网掩码的反码OR运算得到广播地址
5. **子网划分**：根据用户输入的子网数量或每个子网的主机数量进行子网划分
6. **可用IP范围计算**：计算每个子网中可用的IP地址范围

## ⚠️ 注意事项

1. 确保输入的IP地址和CIDR前缀长度格式正确
2. 子网划分时，确保子网数量或每个子网的主机数量合理
3. Web界面需要Python环境和Flask框架支持
4. Windows GUI界面需要Python环境和Tkinter库支持
